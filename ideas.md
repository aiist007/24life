# 倪海厦二十四节气养生历 - 设计理念

## 设计目标
创建一个融合中国传统文化美学与现代交互体验的养生日历网站，以日历形式展示2026年二十四节气的养生指导。

## 方案一：水墨丹青 - 东方禅意美学

### 设计运动
新中式极简主义（Neo-Chinese Minimalism），融合传统水墨画意境与现代数字界面设计。

### 核心原则
1. **留白如画**：大量使用负空间，让内容如同悬浮于宣纸之上
2. **墨分五色**：以黑白灰为主调，点缀五行对应色彩
3. **气韵生动**：通过微妙的动效传达"气"的流动感
4. **虚实相生**：实体内容与虚化背景形成层次对比

### 色彩哲学
- 主色：墨黑(#1a1a1a)、宣纸白(#f8f5f0)、淡墨灰(#6b6b6b)
- 五行点缀色：木青(#4a7c59)、火红(#c53030)、土黄(#d69e2e)、金白(#e8e4dc)、水黑(#1a365d)
- 情感意图：传达宁静、内敛、深邃的东方禅意

### 布局范式
- 竖向卷轴式布局，模拟传统书画的观赏方式
- 左侧固定节气导航条，如同书签
- 右侧主内容区采用不对称留白设计
- 日历以圆形太极图为中心，节气环绕分布

### 标志性元素
1. **水墨晕染过渡**：区块之间使用渐变模糊效果
2. **印章装饰**：节气名称配以红色印章样式
3. **毛笔字体**：标题使用书法风格字体

### 交互哲学
- 悬停时内容如墨迹缓缓显现
- 页面切换如卷轴展开收起
- 点击产生涟漪效果，如水墨滴落

### 动画指南
- 入场动画：内容从透明渐显，如晨雾散去
- 过渡动画：使用ease-out曲线，模拟毛笔收笔
- 微交互：轻微的浮动效果，如气息流动

### 字体系统
- 标题：思源宋体 / Noto Serif SC（700weight）
- 正文：思源黑体 / Noto Sans SC（400weight）
- 装饰：手写书法字体用于节气名称

## 方案二：天圆地方 - 几何宇宙观

### 设计运动
东方几何主义（Oriental Geometry），以八卦、太极等传统符号的几何抽象为设计语言。

### 核心原则
1. **圆方相济**：圆形代表天（时间循环），方形代表地（稳定结构）
2. **对称与平衡**：体现阴阳调和的哲学
3. **层次分明**：通过几何嵌套表达五行相生相克
4. **数理之美**：运用黄金比例和传统度量

### 色彩哲学
- 主色：深靛蓝(#1e3a5f)、暖米白(#faf6f0)、古铜金(#b8860b)
- 五行色：按季节渐变过渡
- 情感意图：庄重、神秘、智慧的宇宙感

### 布局范式
- 中心放射式布局，以年度日历为核心
- 八卦方位决定导航位置
- 节气卡片采用六边形蜂巢排列
- 信息层级通过同心圆环展开

### 标志性元素
1. **太极旋转动画**：页面中心的阴阳鱼缓慢旋转
2. **八卦导航**：八个方位对应八种功能入口
3. **节气轮盘**：24节气围绕中心排列成圆环

### 交互哲学
- 导航如同罗盘转动
- 内容展开如同卦象演变
- 选择产生能量波动效果

### 动画指南
- 入场动画：从中心向外扩散
- 过渡动画：旋转过渡，如天体运行
- 微交互：几何图形的变形动画

### 字体系统
- 标题：霞鹜文楷 / LXGW WenKai（600weight）
- 正文：思源黑体 / Noto Sans SC（400weight）
- 数字：等宽字体，强调节气日期

## 方案三：四时流转 - 自然意象美学

### 设计运动
自然主义数字美学（Digital Naturalism），将四季变化的自然意象融入界面设计。

### 核心原则
1. **顺时而变**：界面色彩随节气自动变化
2. **有机形态**：使用自然曲线替代生硬几何
3. **质感真实**：模拟纸张、木纹、石材等自然材质
4. **生命气息**：通过动效传达万物生长的韵律

### 色彩哲学
- 春：嫩绿(#7cb342)、桃粉(#f48fb1)、天青(#81d4fa)
- 夏：翠绿(#2e7d32)、火红(#d32f2f)、金黄(#ffc107)
- 秋：枫红(#bf360c)、金橙(#ff8f00)、麦黄(#f9a825)
- 冬：霜白(#eceff1)、墨蓝(#1a237e)、松绿(#004d40)
- 情感意图：温暖、亲切、生机勃勃

### 布局范式
- 横向时间轴布局，如四季更替
- 卡片式节气展示，带有自然纹理背景
- 侧边栏显示当前季节养生要点
- 底部固定五行对应信息栏

### 标志性元素
1. **季节背景动画**：飘落的花瓣/树叶/雪花
2. **自然纹理卡片**：每个节气卡片有独特的自然图案
3. **植物装饰边框**：使用花草藤蔓作为装饰元素

### 交互哲学
- 悬停如微风拂过，内容轻轻摇曳
- 点击如种子发芽，信息逐层展开
- 滚动如季节流转，背景渐变过渡

### 动画指南
- 入场动画：从下方生长出现
- 过渡动画：使用spring弹性曲线，如植物摇曳
- 微交互：呼吸效果，如生命脉动

### 字体系统
- 标题：站酷快乐体 / Zcool KuaiLe（用于节气名）
- 副标题：思源宋体 / Noto Serif SC（500weight）
- 正文：思源黑体 / Noto Sans SC（400weight）

## 选定方案：方案三 - 四时流转（自然意象美学）

### 选择理由
1. **用户友好**：自然意象更容易被大众理解和接受
2. **视觉丰富**：四季变化提供丰富的视觉层次
3. **情感共鸣**：自然元素能唤起人们对养生的共鸣
4. **技术可行**：动效和交互在现代前端框架中易于实现
5. **内容适配**：与二十四节气的自然属性高度契合

### 实施要点
- 使用Tailwind CSS实现响应式布局
- 使用Framer Motion实现自然流畅的动画
- 根据当前日期自动高亮对应节气
- 五行色彩贯穿整体设计
- 日历视图与详情视图无缝切换
